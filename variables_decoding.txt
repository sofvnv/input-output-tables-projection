Все переменные, используемые в файле projection_RUS.ipynb:

1. ТАБЛИЦЫ РЕСУРСОВ

- supply2011_codes - базовая таблица ресурсов за 2011 год с кодами для столбцов
- supply2011 - базовая таблица за 2011 год без кодов вообще
 
- agg_supply2011 - итоговый агрегированный вариант базовой таблицы ресурсов за 2011 год
- agg_supply2011_arr - agg_supply2011 в формате numpy array
- agg_supply2011_vec - векторизованный agg_supply2011_arr
 
- rows_sums_agg_supply2011 - сумма по строкам агрегированной таблицы ресурсов за 2011 год
- columns_sums_agg_supply2011 - сумма по столбцам агрегированной таблицы ресурсов за 2011 год
- totals_agg_supply2011 - итоговый вектор ограничений для агрегированной таблицы ресурсов 2011 года (вектор из сумм по строкам и столбцам)
 
 Для 2016 года - аналогично.
 
2. ТАБЛИЦЫ ИСПОЛЬЗОВАНИЯ В ПОТРЕБИТЕЛЬСКИХ ЦЕНАХ
 
- cons_use2011 - агрегированная таблица использования в потребительских ценах за 2011 год

- cons_use2011_arr - cons_use2011 в формате numpy array
- cons_use2011_vec - векторизованный cons_use2011_arr

- rows_sums_cons_use2011 - сумма по строкам агрегированной таблицы использования в потребительских ценах за 2011 год
- columns_sums_cons_use2011 - сумма по столбцам агрегированной таблицы использования в потребительских ценах за 2011 год
- totals_cons_use2011 - итоговый вектор ограничений для агрегированной таблицы использования в потребительских ценах за 2011 год

Для 2016 года - аналогично.

3. ТАБЛИЦЫ ИСПОЛЬЗОВАНИЯ В ОСНОВНЫХ ЦЕНАХ

- basic_use2011 - агрегированная таблица использования в основных ценах за 2011 год

- basic_use2011_arr - cons_use2011 в формате numpy array
- basic_use2011_vec - векторизованный cons_use2011_arr

- rows_sums_basic_use2011 - сумма по строкам агрегированной таблицы использования в потребительских ценах за 2011 год
- columns_sums_basic_use2011 - сумма по столбцам агрегированной таблицы использования в потребительских ценах за 2011 год
- totals_basic_use2011 - итоговый вектор ограничений для агрегированной таблицы использования в потребительских ценах за 2011 год

Для 2016 года - аналогично.
